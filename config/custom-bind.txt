# Закрепление домена за конкретным IP — статический A-ответ (policy.ANSWER в Knot Resolver)
#
# Полезно, когда нужно жёстко зафиксировать домен за рабочим IP, минуя DNS
# (например, гео-DNS отдаёт IP из заблокированной подсети, а конкретный IP рабочий).
# Домен остаётся на ПРЯМОМ маршруте (НЕ через VPN).
#
# Формат записи: <домен> <IP[,IP2,...]>
# Где:
#   домен - доменное имя в ASCII/Punycode (домен и все его поддомены)
#   IP    - IPv4-адрес (несколько — через запятую или пробел)
#
# Примеры записи:
#   example.com            151.101.3.6
#   sub.example.com        151.101.3.6,151.101.67.6
#
# Приоритет: custom-bind.txt > custom-upstream.txt > маршрутизация AntiZapret
# Строки начинающиеся с # это комментарии и они не обрабатываются
#
